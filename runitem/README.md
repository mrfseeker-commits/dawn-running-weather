# 런닝 복장 추천 시스템

기온, 습도, 풍속 데이터를 기반으로 적절한 런닝 복장을 추천해주는 데이터베이스 프로그램입니다.

## 주요 기능

- 기상 조건(기온, 습도, 풍속)에 따른 런닝 복장 추천
- **극한 기온 안전 경고**: 29°C 이상 또는 -7°C 이하 시 실내 운동 권장
- 한국 러닝 커뮤니티 및 스포츠 브랜드 데이터 기반 (15개 추천)
- **기상 정보 크롤링 프로그램 연동 인터페이스 제공**
- CLI 인터페이스

## 빠른 시작

### 1. CLI 프로그램 실행
```bash
python main.py
```

### 2. 기상 크롤링 프로그램 연동
```python
from weather_interface import WeatherInterface

interface = WeatherInterface()
result = interface.get_outfit_recommendation({
    'temperature': 18.5,
    'humidity': 65,
    'wind_speed': 3.2
})
print(result['recommendations'][0]['top'])
```

**상세 가이드**: [INTEGRATION_GUIDE.md](INTEGRATION_GUIDE.md) 참조

### 필요 조건
- Python 3.6 이상
- 추가 라이브러리 불필요 (표준 라이브러리만 사용)

## 사용 방법

프로그램을 실행하면 다음과 같은 메뉴가 표시됩니다:

1. **복장 추천 받기**: 현재 기상 조건을 입력하여 적절한 복장 추천을 받습니다
2. **모든 복장 데이터 보기**: 데이터베이스에 저장된 모든 복장 데이터를 확인합니다
3. **새로운 복장 데이터 추가**: 새로운 복장 추천 데이터를 추가합니다
4. **복장 데이터 삭제**: 기존 복장 데이터를 삭제합니다
5. **샘플 데이터 초기화**: 기본 샘플 데이터로 데이터베이스를 초기화합니다

## 데이터베이스 구조

### outfit_recommendations 테이블

| 컬럼명 | 타입 | 설명 |
|--------|------|------|
| id | INTEGER | 기본 키 |
| temp_min | REAL | 최저 온도(°C) |
| temp_max | REAL | 최고 온도(°C) |
| humidity_min | REAL | 최저 습도(%) |
| humidity_max | REAL | 최고 습도(%) |
| wind_speed_min | REAL | 최저 풍속(m/s) |
| wind_speed_max | REAL | 최고 풍속(m/s) |
| top | TEXT | 상의 |
| bottom | TEXT | 하의 |
| accessories | TEXT | 액세서리 |
| notes | TEXT | 참고사항 |

## 샘플 데이터

한국 러닝 커뮤니티 및 스포츠 브랜드(나이키 코리아, MO Sports 등)의 추천을 기반으로 한 15개의 상세 데이터를 포함합니다:

### 기온별 추천
- **극한 더위** (29°C 이상): ⚠️ 실내 운동 권장 (열사병, 탈수 위험)
- **한여름** (25-29°C): 싱글렛/민소매, 반바지
- **초여름/초가을** (20-25°C): 반팔, 반바지
- **선선한 날씨** (15-20°C): 반팔 또는 얇은 긴팔, 반바지
- **쌀쌀한 날씨** (10-15°C): 얇은 긴팔, 타이즈 또는 반바지
- **추운 날씨** (7-10°C): 얇은 긴팔 (+조끼), 얇은 장갑
- **본격 추위** (5-7°C): 긴팔+조끼/재킷, 보온 장갑
- **매우 추운 날씨** (0-5°C): 2-3겹 레이어링 (조끼 활용), 방한 장갑
- **한겨울** (-7~0°C): 3겹 레이어링, 방한 장비
- **극한 추위** (-7°C 이하): ⚠️ 실내 운동 권장 (동상, 저체온증 위험)

### 특수 조건
- 습도와 풍속에 따른 세부 추천 포함
- 고습도 환경에서는 속건성 소재 강조
- 강풍 시 방풍 기능 장비 추가 권장

## 예제

### 일반 복장 추천

```
기온(°C): 18
습도(%): 65
풍속(m/s): 3

추천 복장:
상의: 반팔 티셔츠 또는 얇은 긴팔
하의: 러닝 반바지
액세서리: 암슬리브 (필요시)

```

```
기온(°C): 32
습도(%): 70
풍속(m/s): 3

상의: 실외 런닝 부적절
하의: 실내 운동 권장

현재 기온 32°C - 매우 더운 날씨로 열사병, 탈수 위험이 높습니다.
실내 트레드밀이나 에어컨이 있는 체육관에서 운동하시거나,
이른 아침(5-7시) 또는 늦은 저녁(20-22시) 시간대를 이용하세요.
```
